Las compus y los dados

Las compus hacen lo que se les dice, cuando se les dice; exactamente como se les dice. No tienen voluntad.

Se hace mucho por que sean predecibles hasta la nausea. Si se les pide lo mismo, responderán lo mismo una y otra vez. Cualquier otro resultado sería un error.

Por eso mismo son malas para los dados: Es difícil hacer que una compu entregue un número aleatorio.

Para dar la ilusión de aleatoriedad se usa información que está cambiando (fuentes de entropía):

  • la hora del día,
  • la temperatura del procesador,
  • la posición del puntero,
  • el número de proceso,
  • el tiempo de ejecución…

…y se combina intentando maximizar la variabilidad del resultado. Para esto se pueden usar procesos similares a los de los resúmenes criptográficos, cuyo resultado cambia mucho si se les da información casi igual.

De esta forma, la compu puede darnos resultados parecer aleatorio.

Pero cuidado: esos dados están cargados.
