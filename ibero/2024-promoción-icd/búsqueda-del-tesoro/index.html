¿Cómo puedo agregar un formulario en wordpress donde si me dan la respuesta correcta le muestro algo al usuario y en caso de respuesta incorrecta le muestro otra cosa?

Para agregar un formulario en WordPress que muestre diferentes contenidos según la respuesta del usuario, puedes seguir estos pasos:

Usando un Plugin ↑

  1. Instalar un plugin como Contact Form 7 o WPForms:

    • Ve a Plugins > Añadir nuevo.
    • Busca Contact Form 7 o WPForms y haz clic en Instalar y luego en Activar.

  2. Crear el formulario:

    • Si usas Contact Form 7, crea un nuevo formulario en Contact > Add New.
    • Si usas WPForms, ve a WPForms > Add New.

  3. Agregar el campo de pregunta:

    • Añade un campo de texto o una opción select para la pregunta.

  4. Configurar Redirección o Mensaje de Éxito/Error:

    • Configura la acción dependiendo de la respuesta. Puedes usar JavaScript personalizado para validar la respuesta y mostrar un mensaje.

Ejemplo con JavaScript Personalizado y Contact Form 7 ↑

<!-- Contact Form 7 Shortcode -->
[text* quiz_answer "Respuesta"]
[submit "Enviar"]

<!-- JavaScript personalizado -->
<script>
document.addEventListener('wpcf7mailsent', function(event) {
    var answerField = document.querySelector("input[name='quiz_answer']");
    var userAnswer = answerField.value.toLowerCase().trim();
    var correctAnswer = "respuesta correcta";

    if (userAnswer === correctAnswer) {
        // Muestra contenido si la respuesta es correcta
        alert("¡Correcto! Aquí tienes el contenido especial.");
    } else {
        // Muestra un mensaje de error si la respuesta es incorrecta
        alert("Respuesta incorrecta. Inténtalo de nuevo.");
    }
}, false);
</script>

Usando Código Personalizado ↑

Paso 1: Crear un Formulario ↑

<form id="quiz-form">
    <label for="quiz-answer">¿Cuál es la capital de Francia?</label>
    <input type="text" id="quiz-answer" name="quiz-answer" required>
    <input type="submit" value="Enviar">
</form>
<p id="message"></p>

Paso 2: Agregar el Script de Validación ↑

<script>
document.getElementById('quiz-form').addEventListener('submit', function(event) {
    event.preventDefault();

    var userAnswer = document.getElementById('quiz-answer').value.toLowerCase().trim();
    var correctAnswer = "paris";
    var messageElement = document.getElementById('message');

    if (userAnswer === correctAnswer) {
        messageElement.textContent = "¡Correcto! Aquí está tu contenido especial.";
        // Aquí puedes agregar contenido HTML, redirigir a otra página, etc.
    } else {
        messageElement.textContent = "Respuesta incorrecta. Inténtalo de nuevo.";
    }
});
</script>

Coloca el HTML y el JavaScript en la página o publicación donde desees mostrar el formulario.

¿Cómo darle la URL a wp_redirect?

Puedes indicar la URL completa con wp_redirect, por ejemplo: wp_redirect('https://www.ejemplo.com/pagina');.

También puedes construir dinámicamente la URL con variables PHP, como por ejemplo: $pagina = home_url('/ruta'); wp_redirect($pagina);.

Otra opción es utilizar la constante SITE_URL para simplificar la tarea: wp_redirect(SITE_URL . '/ruta');.

Y por último, puedes añadir parámetros GET a la URL, como en: wp_redirect(add_query_arg('parametro', 'valor', home_url()));.
