  • Convertir los códigos postales a coordenadas.

    Dado que los estudiantes llenan manualmente las colonias, los datos deben limpiarse y luego usar un mecanismo de fuzzy matching.

    Como esto llevaría mucho tiempo, se utiliza el centroide de los datos de coordenadas, calculado con norma L1 para latitud y longitud.

    Si los datos deben evaluarse de forma más precisa se puede hacer la limpieza.
