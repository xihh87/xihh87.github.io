COMO conectar dos computadoras que usan linux via ethernet

Como demostración de que las computadoras están diseñadas para conectarse de par a par:

COMO conectar dos computadoras que usan linux usando un cable ethernet ↑

Después de conectar el cable ethernet en las dos, abrir terminale en una y escribir:

sudo ifconfig eth0 10.0.0.1

Y en la otra:

sudo ifconfig eth0 10.0.0.2

Listo, ahora pueden establecer una conexión entre sí, se puede comprobar escribiendo desde alguna la IP de la otra:

ping 10.0.0.X

Donde X es «1» o «2» según corresponda.

Aunque en realidad sólo es interesante si instalan un servidor web o ftp, en cuyo caso se podrá ver la página web.
