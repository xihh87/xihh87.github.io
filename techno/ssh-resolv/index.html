Auto-resolución de dominios en ssh

Por seguridad, el cliente ssh no completa los dominios como los otros programas, sino que pide la configuración explícita de los dominios que debe resolver.

Para corregir este problema, se agrega a ~/.ssh/config:

Host *
	CanonicalizeHostname	yes
	CanonicalDomains	dominio.a.resolver	otro.dominio.com	[...]

Con esto se resuelve el problema.
