Notación de Modelado de Procesos de Negocio (Business Process Model and Notation) ↑

La especificación oficial de BPMN y un ejemplo de uso se pueden consultar en línea, hay un resumen de su sintaxis en Wikipedia.

  • PlantUML debería generar BPMN, aunque aún no los implementan, Son similares a los diagramas de actividad.

  • BPMN Sketch Miner es una herramienta web para minar procesos en sesiones en vivo. tienen un paper, los ejemplos no muestran algo muy distinto de PlantUML.

  • bpmn-js es una herramienta web para dibujar procesos en BPMN, está bonita pero usa principalmente clicks. Escribir XML a mano es tortura.

Entidad relación ↑

  • Plantuml puede generar diagramas ER. No me terminan de gustar.

  • Mermaid puede hacer varios tipos de diagramas a partir de texto, incluyendo diagramas entidad relación, aunque el diagrama de datos JSON se parece más a lo que me gustaría de un diagrama ER.

  • Graphviz es un sistema muy versátil (y por lo tanto complejo) para representar grafos. Usando la sintaxis de nodos que representan registros se pueden generar diagramas entidad-relación. (sintaxis, formas de nodos, custom forms)

En algunos casos conviene generar los diagramas a partir de la base de datos.

Mapa mental ↑

  • Mermaid es una aplicación para hacer diagramas a partir de texto que puede hacer mapas mentales.

  • Markmap.js (vía mindmapexpert, demo) Usa una sintaxis similar a markdown y hace mapas interactivos. Tiene plugin para VS Code y cli.

  • butterfly biblioteca de alibaba para hacer varios diagramas interactivos, literalmente “está en chino”.
