Flujos de trabajo ↑

Con ayuda de k054 y Dawid Oliva encontré cómo hacer diagramas de los procesos que automatizo.

El formato debe ser de la siguiente forma:

digraph {
	anotation -> vcf_phasing;
	vcf_phasing -> split_vcf_by_sample;
	split_vcf_by_sample -> variant_calling;
	variant_calling -> base_recal;
	base_recal -> realign;
	realign -> indexbam;
	indexbam -> markdup;
	markdup -> merge_by_sample;
	merge_by_sample -> sort;
	sort -> sambam;
	sambam -> alignment;
	alignment -> trimmomatic;
	trimmomatic -> init;
}

Y se compila con el comando:

dot -Tpng archivo.dot > diagrama.png

Queda pendiente hacer el script que toma los mkfile y los convierte.
