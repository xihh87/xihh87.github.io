manubot es un programa para gestionar referencias en artículos.

El programa está muy acoplado a la plantilla del proyecto pero tienen una interfaz multipropósito llamada pandoc-manubot-cite que puede usarse en cualquier lugar.

Agregar manubot a un proyecto ↑

Agregar a tu script para generar documentos:

# file: cmd/build-document
pandoc --filter=pandoc-manubot-cite --citeproc ${ARGS}

Agregar referencias en un texto ↑

En el contenido de tu documento agregas:

Este es un libro [@isbn:0-19-852663-6],
esta es una web [@url:https://sitio/web/],
este es un preprint [@arXiv:1905.09749].

Agregar bibliografía en un texto sin referenciarla ↑

Este script para generar “Awesome lists” usa archivos con una referencia por cada línea y las utiliza en la línea de comando de manubot render.

Generar alias para referencias ↑

Agregar a los metadatos de tu archivo:

---
citekey-aliases:
  nombre1: "isbn:0-19-852663-6"
  nombre2: "url:https://sitio/web/"
  nombre3: "arXiv:1905.09749"
---

Corregir las referencias ↑

En ocasiones la referencia que trae manubot contiene algún error desde origen de la fuentes que consulta para arreglarte la vida. En esos casos:

  1. Generar la bibliografía con manubot:

    manubot cite arXiv:1905.09749 > biblio.json

    Cuando manubot no puede encontrar la bibliografía, la suelo generar usando Zotero con la opción Exportar elemento > Formato: CSL JSON. Esto es poco común.

  2. Editar el archivo generado:

    [
  {
    "id": "arXiv:1905.09749",
    // debes eliminar la siguiente línea:
    "note": "OCLC: 1107483160\nThis CSL JSON Item was automatically generated by Manubot v0.4.0 using citation-by-identifier.\nstandard_id: isbn:9781259588099",
    // debes verificar que el json siga siendo válido después de tus cambios
    // eso a veces significa eliminar o agregar las ","
    […]
  }
}

    Es importante borrar la nota que genera manubot porque manubot ignora la bibliografía generada por manubot.

    Verificar tu archivo bibliográfico con jq:

    jq . biblio.json

  3. Agregar la bibliografía a tu artículo:

    ---
bibliography:
  - biblio.json
---

Cambiar el formato de las referencias ↑

Si necesitas tus referencias en un formato distinto al que da por defecto puedes descargar un CSL y agregar a las opciones de pandoc:

--csl=${DEFINICIÓN_DE_FORMATO_BIBLIOGRÁFICO}.csl
