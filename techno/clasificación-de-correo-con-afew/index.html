Como quiero dejar la mala costumbre de utilizar gmail estuve revisando la alternativa hacker de Daniel Patterson.

La idea es instalar en tu propio servidor:

  • qmail para entregar y recibir correo,
  • dovecot para leer el correo recibido,
  • spamassassin para evitar el correo basura,
  • notmuch para realizar búsquedas y
  • afew para clasificar el correo.
  • mmh para componer el correo.
