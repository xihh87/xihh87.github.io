Utox

Es un software para conversar de manera segura y fácil.

Instalación ↑

Para instalar el software en ubuntu/debian/etc se puede copiar y pegar este código en una terminar de root:

sudo sh -c 'echo "deb https://repo.tox.im/ nightly main" > /etc/apt/sources.list.d/tox.list'
wget -qO - https://repo.tox.im/pubkey.gpg | sudo apt-key add -
sudo apt-get install apt-transport-https
sudo apt-get update -qq
sudo apt-get install utox

El código agrega el repo de los desarrolladores y lo instala desde allí.

En windows se puede descargar qTox aquí.

uTox ↑

Al abrir uTox, se muestra la siguiente ventana. Aquí se pueden añadir los ID de las personas con quienes conversaremos.

En el menú opciones se puede cambiar el nombre y copiar el ID.

Para conversar con alguien hay que enviarle el ID para que lo agregue a su cliente de tox.
