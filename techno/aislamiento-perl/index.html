Construir paquetes aislados de perl

Como varios programas, perl usa variables de entorno para buscar sus librerías.

El módulo de perl local::lib permite hacer paquetes aislados.

PKGDIR=/pkg/prinseq-lite/0.20.4/x86_64
eval $(perl -I$PKGDIR/lib/ -Mlocal::lib=$PKGDIR)

Para terminar la sesión:

eval $(perl -Mlocal::lib=--deactivate,~/path)

#!/usr/bin/env PERL5LIB=/pkg/prinseq-lite/0.20.4/x86_64/lib/perl5 PERL_LOCAL_LIB_ROOT=/pkg/prinseq-lite/0.20.4/x86_64/ /usr/bin/perl
