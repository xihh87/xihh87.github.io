Importar llave ↑

Importar las llaves con pacman-key tiene un poco de truco.

Hay que generar una carpeta con un archivo llamado pubring.gpg:

mkdir -p pacman-key-export
gpg --export > pacman-key-export/pubring.gpg

Una vez que tenemos la llave en ese formato, se puede importar usando:

sudo pacman-key import pacman-key-export
