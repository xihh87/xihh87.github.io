Hakyll es una librería para generar sitios de contenidos estáticos, similar a HUGO pero que usa pandoc.

Hakyll en debian ↑

Hakyll no está disponible en los repos de debian. Por eso instalaremos primero stack (tampoco en los repos de debian).

Instalar stack en debian ↑

sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 575159689BEFB442
echo 'deb http://download.fpcomplete.com/debian jessie main'|sudo tee /etc/apt/sources.list.d/fpco.list
sudo apt-get update && sudo apt-get install stack -y

Instalar hakyll en debian ↑

Usando stack se instala hakyll.

stack setup --resolver lts-2.15 7.8.4
stack update
stack install --resolver lts-2.15 hakyll

Las últimas versiones del resolver de stack no pueden compilar hakyll. Por eso recurrimos a la versión 2.15.

Hakyll en Arch ↑

Se puede instalar hakyll de los repos de Arch Haskell agregando a /etc/pacman.conf:

# /etc/pacman.conf

[haskell-core]
Server = http://xsounds.org/~haskell/core/$arch

[haskell-happstack]
Server = http://noaxiom.org/$repo/$arch

Para instalar cosas con un ghc viejo, debe instalarse ncurses5-compat-libs desde AUR de acuerdo con el manual de stack.

Generar el sitio web ↑

hakyll-init [nombre]
cd [nombre]
stack init
stack build --install-ghc --resolver=lts-2.15
stack exec site build
