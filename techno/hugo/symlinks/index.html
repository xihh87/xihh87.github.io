Mi web dejó de funcionar

Mi web dejó de funcionar de un día para otro en una actualización. Sin aviso. Sin opción de configuración.

La razón es que dejaron de permitir el uso de enlaces simbólicos (y no van a volver) para reemplazarlos con un mecanismo que llaman montado, la idea es que permiten tener varios orígenes para un mismo directorio y tal vez otras ventajas.

Todo como parte de un proceso de migración a módulos.

Sin embargo, los montajes no siempre funcionan (exhibit A).

Aquí hay una guía para diagnosticar los montajes: puedes consultar el grafo de dependencias de módulos, y mostrar la configuración de montaje.

Hay quienes usan otros métodos para incluir archivos.
