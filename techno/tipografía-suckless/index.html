Elegir fuente ↑

Un problema que tuve es que no sabía cómo elegir la fuente, que debe especificarse de acuerdo a la sintaxis de fonts.conf.

Encontré un programa que muestra una lista de las fuentes instaladas y su nombre en ese formato:

fc-list

Lo dejo aquí para recordarlo y por si le puede servir a alguien.

Cómo saber qué caracteres soporta una tipografía ↑

Al parecer este script permite revisar los caracteres:

ls-chars "Tipografía"

Si eso no funciona, se pueden descifrar los jeroglíficos de fc-match.

No encontré un método para buscar una tipografía que contenga un caracter.
