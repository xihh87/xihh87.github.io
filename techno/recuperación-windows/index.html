Recuperación de Windows

Un sistema Windows cargaba demasiadas cosas al iniciar sesión y no terminaba de iniciar correctamente.

Lo primero que hago en estos casos es respaldar la información desde un usb live de linux, pero en este caso, el sistema de archivos estaba dañado y sólo se puede revisar desde Windows.

Iniciar el sistema de revisión de disco ↑

  • Lanzar el administrador de tareas con <Ctrl> <Alt> <Supr>.

  • Teclear F10 y buscar la opción Archivo > Nueva Tarea....

  • La nueva tarea es cmd y hay que ejecutarla en modo de privilegios elevados.

  • Desde cmd ejecutar shutdown /r /o para reiniciar en modo avanzado.

  • Elegir de nuevo Símbolo de sistema.

  • Iniciar sesión.

  • Ejecutar chkdsk /f c:.

  • Para salir escribir exit.

Respaldar los datos ↑

  • Iniciar con un sistema vivo de linux y hacer rsync hacia un disco duro externo.

Recuperar windows ↑

  • Lanzar el administrador de tareas con <Ctrl> <Alt> <Supr>.

  • Teclear F10 y buscar la opción Archivo > Nueva Tarea....

  • La nueva tarea es cmd y hay que ejecutarla en modo de privilegios elevados.

  • Desde cmd ejecutar shutdown /r /o para reiniciar en modo avanzado.

  • Elegir restablecer el equipo.

Iniciar windows desde un grub sin configuración ↑

grub> chainloader /efi/Microsoft/Boot/bootmgfw.efi
grub> boot

Recuperar el sistema de arranque de Windows en EFI ↑

cmd> bcdboot.exe C:\Windows
