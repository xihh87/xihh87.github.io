Inicio de sesión automático

Como mi sistema operativo y mi directorio home están cifrados, cada que inicia mi computadora tengo que usar una contraseña, y para iniciar el sistema tengo que usar otra.

Prefiero configurar mi computadora para auto iniciar sesión.

Al parecer esto es mucho más sencillo en Arch que en otras distribuciones, porque hay menos partes móviles y las configuraste tú mismo.

Esta misma configuración no necesariamente funciona en otras distribuciones, aunque el procedimiento para Ubuntu es similar.

Automaticé este proceso en este repositorio usando el rol auto_login.

  1. Pedir a la consola para iniciar sesión de X en la consola 1 automáticamente.

    # https://wiki.archlinux.org/title/Xinit#Autostart_X_at_login
if [ -z "${DISPLAY}" ] && [ "${XDG_VTNR}" -eq 1 ]; then
  exec startx
fi

  2. Configurar el inicio de sesión para el teletipo correspondiente.

    $ sudo systemctl edit getty@tty1
[Service]
ExecStart=
ExecStart=-agetty --autologin x --noclear %i $TERM

    Esta opción sobrescribe una parte de la configuración.

Hay quien recomienda usar directamente un programa en C para iniciar sesión con el sistema:

int main()
{
  execlp("login", "login", "-f", "usuario", 0);
}
