Cómo generar RSS usando docpad

RSS es un estándar para anunciar los cambios de un sitio web.

Agregar un RSS a Docpad es muy simple:

  • Instalar el complemento usando docpad install rss

  • Generar una colección de documentos en docpad.coffee

    collections: # Contiene los documentos en posts. posts: -> @getCollection(‘documents’).findAllLive({relativeOutDirPath: ‘posts’},[{date:-1}])

  • Agregar a la configuración de docpad en la sección plugins

    plugins: rss: default: collection: ‘posts’ url: ‘/rss.xml’

Después esperamos a que docpad genere los archivos o los regeneramos usando:

docpad generate

Finalmente agregamos a nuestras plantillas:

<link rel="alternate" type="application/rss+xml" title="aquí va el título" href="/rss.xml" />

Y para tener un botón de suscripción dentro de tu sitio web:

<a rel="alternate" type="application/rss+xml" title="aquí va el título" href="/rss.xml">
  <img src="ruta/a/la/imagen/rss.png"/>
</a>
