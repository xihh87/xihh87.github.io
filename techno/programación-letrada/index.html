Programación letrada

Dado que la programación es literatura práctica es mucho más útil cuando enseña a un ser humano cómo resolver el problema.

Esto se puede entender en varios niveles:

  • Una explicación del problema y la solución propuesta.
  • Las instrucciones para utilizar el programa para resolver el problema.
  • La documentación de cómo funciona el programa.

Programar documentando el código es costoso, pero obliga a explicar el diseño y facilita la cooperación
