Instalar packrat ↑

Desde la consola de R:

install.packages("packrat")

Iniciar un proyecto ↑

packrat::init("~/projects/babynames")

Una vez iniciado el proyecto, cada vez que se ejecute R desde ese directorio usará la librería de packrat que está instalada allí.

https://rstudio.github.io/packrat/walkthrough.html

Instalar paquetes ↑

install.packages("reshape2")
packrat::status()
packrat::snapshot()

Instalar paquetes locales ↑

Para acceder a código local:

packrat::set_opts(local.repos = "./libs")
packrat::install_local("nombre.paquete")
