¿Se puede usar múltiples EFI?

Tengo una computadora con Windows y

  • no puedo desarrollar en ella correctamente
  • es diferente a mi servidor productivo.

Intenté instalar Ubuntu y funciona correctamente, pero sólo tengo 100MB en la partición EFI, y para instalar el firmware de la tarjeta gráfica necesito más de los 18MB libres.

Es muy probable que si hago la partición de EFI más grande Windows deje de iniciar. Esto alguna vez me ocurrió en Windows 7.

Pensé que tal vez podría usar un EFI en un disco diferente para cada sistema sistema operativo pero parece que la forma recomendada es usar una única ESP.

Supuestamente sí es posible utilizar varias EFI, sin embargo yo no lo he logrado. Sospecho que los BIOS toman la primer partición y usan esa, independientemente de que se registren varias.

Se tiene que generar la partición, se tiene que registrar como una entrada de EFI (puede hacerse con efibootmgr).

Para resolver el problema:

  1. Hice una partición EFI en el segundo disco duro.

  2. Copie todos los archivos de la EFI original a esta nueva EFI.

  3. Desactivé las opciones boot,esp en la EFI original usando gparted.

De esta manera:

  • no se utiliza la partición EFI original sino mi nueva partición EFI.

  • tengo la posibilidad de regresar la compu con EFI de windows dejo la partición EFI original sin cambios.

  • ahora tengo espacio suficiente para agregar firmware y actualizar.
