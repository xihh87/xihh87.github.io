Cómo anotar con hypothesis en tu sitio web ↑

Para empezar a usar anotación en tu web puedes agregar al encabezado de tu web:

<script src="https://hypothes.is/embed.js"></script>

Cómo anotar cualquier sitio web ↑

Si quieres comentar una web que no incluye este script, también puedes ejecutar el código directamente desde la barra de direcciones del navegador:

javascript:(
  function(){
    window.hypothesisConfig=function(){
      return{showHighlights:true,appType:'bookmarklet'};
    };
    var d=document, s=d.createElement('script');
    s.setAttribute('src','https://hypothes.is/embed.js');
    d.body.appendChild(s)
  }
)();

Que agrega ese mismo código en cualquier página o PDF que consultes en tu navegador.

Ese código puede guardarse en tus favoritos y agregarle una palabra clave.

Si usas chrome puedes descargar su plugin.

Otra manera de hacerlo es visitar la página a través del proxy de hipothesis:

https://via.hypothes.is/${URL}

Dónde se guardan tus anotaciones ↑

Usando la configuración por defecto, las anotaciones se guardan en los servidores de hypothes.is en una instancia del software h.

Si quisieras, podrías montar y mantener tu propio servicio y tendrías que apuntar la fuente del código hacia tu servidor de anotaciones.

El servicio parece respetar tu privacidad ↑

Como parte del servicio se entrega un correo para generar la cuenta y se asocian los comentarios a esa cuenta.

Pero argumentan que para usar el servicio no expones información personal y por lo tanto no necesitan un anuncio de uso de cookies.

Cómo diferenciar las anotaciones en diferentes versiones del texto ↑

Escribo mis textos en formato de pandoc y voy editando las versiones de mi texto.

Quisiera que las anotaciones se apliquen a la última versión del texto publicada.

Sin embargo, parece que no hay forma de anotar las versiones si no están incluidas en la URL.

Se supone que hypothes.is usa anclas robustas para sus comentarios y puede acomodar los comentarios si el texto se mueve cuando puede identificar el contenido
