Registro de una sesión de terminal

abduco es un programa simple que permite mantener programas corriendo aunque no tengamos la terminal abierta.

Es un poco como tmux o screen pero sólo tiene la funcionalidad de mantener la sesión y usa dvtm para mostrar muchos programas a la vez y las demás monadas de screen.

Y el programa puede usarse para conectarse al output de la sesión, que puede guardarse utilizando una redirección simple:

SESION=nombre-sesion
abduco -n $SESION; abduco -r $SESION > $SESION.log &

Luego, una sesión puede usarse y será registrada automáticamente:

abduco -a $SESION

El log es un archivo binario que muestra la salida de la terminal. Todavía no resuelvo cómo leerlo si se hace limpieza de pantalla y etc, pero si no hay salidas especiales se puede ver el log usando:

cat $SESION.log
