run s/ command on nvim on all files

To run a substitution command on all files in Neovim, you can use the argdo command along with the :%s/old/new/g command. Here’s how you can do it:

:args **/*.txt | argdo %s/old/new/g | update

This command will apply the substitution s/old/new/g on all .txt files in the current directory and its subdirectories.

Another way is to use :cfdo which targets all files in the quickfix list:

:cfdo %s/old/new/g | update

This command will run the substitution on every file in the quickfix list.

You can use the ** wildcard to target all files in the current directory and its subdirectories.

:args ** | argdo %s/old/new/g | update

Remember to replace old with the text you want to replace and new with the text you want to replace it with.
