Estoy trabajando en un curso de bioinformática y quiero automatizar varias cosas.

Tengo varias ideas que no quiero que se pierdan en el olvido:

  • Automatizar la progresión del curso de git usando los minutos gratuitos de GitLab y pruebas unitarias con un docker que contenga los datos.

    Si el código de los estudiantes funciona, enviar a la plataforma que tenga una señal de que pasó el ejercicio y configurar el repositorio para el siguiente ejercicio.

  • Para los mensajes con los estudiantes, tener un script que cambie variables por datos de los alumnos.

    Estoy pensando que la forma más simple con execline es, para cada estudiante:

    1. Guardar los datos en archivos como envdirs en un directorio por defecto.

    2. Generar un directorio para tarjetas que pueda usar como ${XDG_TEMPLATE_DIR}.

    3. Con n elegir el mensaje y con un pipe cambiar las cosas que deban cambiar.
