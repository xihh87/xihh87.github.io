Si guardaste tu archivo en git y lo borraste puedes recuperarlo así

  • Archivos borrados o versiones anteriores de tu archivo
  • Referencias borradas

Archivos borrados o versiones anteriores de tu archivo ↑

Puedes recuperar archivos que guardaste en git usando:

git checkout ${commit} -- ${archivo}

Referencias borradas ↑

Si borras accidentalmente referencias, se pueden listar usando git fsck --full y se pueden recuperar con el comando:

git checkout -b ${nombre} ${commitid}

Si esto acaba de pasar, muchas veces puedes ver el commitid con el comando:

git reflog
