Usar libvirt en archlinux

La configuración por defecto de libvirt usa polkit para iniciar sesión, y si no tienes un agente de Polkit en tu sesión fallaría con el error:

Call to virConnectOpen failed: authentication unavailable: no polkit agent

La solución es agregar al usuario al grupo libvirt:

# usermod --apend --groups libvirt $(whoami)
# gpasswd -a $(whoami) libvirt
