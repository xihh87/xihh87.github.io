Github Co-Pilot ↑

Este modelo se utiliza para dar un servicio para ayudarte a programar. Probé este modelo y me pareció que:

  • Muchas veces te evita escribir cosas repetidas.
  • Generar plantillas de código que generalmente contiene errores para corregir y no siempre es lo que buscas.
  • Genera código más complejo de lo que se necesita.
  • Muchas veces se inventa nombres de parámetros en configuraciones.

No me llama la atención lo suficiente para pagar este servicio en concreto.

ChatGPT ↑

Probé la beta de ChatGPT y es el primer modelo generador de lenguaje y código que entrega cosas parecidas a lo que espero.

Para probar ChatGPT:

  • Registrarse en la página oficial del modelo.
  • Cuándo se apruebe la solicitud entrar a la misma página y preguntarle cosas.

Al parecer puede hacer muchas cosas interesantes como redactar correos, enseñarte conceptos, resolver examenes, fingir ser una terminal, pero dice muchas mentiras (otro supuesto ejemplo).

Tabnine ↑
