Encontré una aplicación increíble que no funciona para mi ↑

Hace un tiempo vi Chatdev y me pareció genial la idea del desarrollo de software automatizado, pero me decepcionó que no logró hacer las herramientas que le pedí.

Tampoco me gustó que:

  • Se trabó en uno de los proyectos porque hizo más peticiones de las permitidas.
  • Tiene que empezar un proyecto de cero y no puede corregir.

Por lo tanto, me gustaría hacer mi propio chatdev que pueda hacer pedacitos de desarrollo automatizado.

Por eso adapté GPT a mi forma de trabajo ↑

Por eso hice una interfaz de GPT que funciona como filtro de unix.

(Como fabric, antes de saber que fabric existe, más simple pero con capacidad de visión y generar imágenes con dall-e.)

Tengo ahora los siguientes roles:

  • py genera código en python a partir de la descripción o función.
  • pyn genera código python delimitado por ```{python} para usar en notebooks.
  • pydoc genera documentación de código a partir de la función.

Estos prompts con mi interfaz cumplen lo que para otros co-pilot pero únicamente bajo demanda.

Pero estoy buscando convertirlo en mi becario ↑

Me gustaría tener un sistema de agentes que me ayuden con:

  • Revisión de código en función de ciertos criterios de funcionalidad, calidad y seguridad.

Este es el plan ↑

  • Generar una interfaz que le permita a GPT leer archivos de la compu y guardar cambios en git (commit cb45cd21).
    • Agregar una interfaz que le permita describir los cambios y justificarlos.
  • Generar un ciclo de retroalimentación para revisar código.
    • Ejecutar las pruebas unitarias.
    • Generar el prompt con el código, el resultado de las pruebas.
