Notas acerca de YAML

La especificación oficial de YAML en 2021 es la tercera revisión (1.2).

El formato tiene varios problemas y casos que podrían fallar de maneras inesperadas, por lo que se dice que YAML “es inseguro por defecto”.

Por esa razón, se recomienda usar un subconjunto de YAML más estricto.

Parece que se puede utilizar filebeat para verificar un archivo

$ FILE=.yml
$ filebeat test config -c ${FILE}

Pero sería mejor usar un verificador dedicado para yaml:

$ yamllint ${FILE}
