Generar historial de versiones de archivos binarios.

Para configurar git para ver archivos binarios, hay que usar un programa para convertirlos en texto:

FILETYPE=
COMMAND=
git config --global diff.$FILETYPE.textconv $COMMAND

Agregar $HOME/.config/git/attributes el nombre de los archivos y su tipo:

printf "*.$FILETYPE\tdiff=$FILETYPE" >> $HOME/.config/git/attributes
