Programas distintos utilizan métodos distintos para saber con qué abrir páginas web.

El programa necesita tener un archivo desktop, que dice lo que se puede abrir con él. El archivo del fabricante generalmente está en ´/usr/share/applications/´ y puede modificarse para cada usuario en $HOME/.local/share/applications/.

Hay un archivo que dice qué aplicaciones deben abrir cada cosa para cada usuario: $HOME/.local/share/applications/mimeapps.list

Para editarlo, se puede usar el programa xdg-mime

´´´ $ xdg-mime default browser.desktop x-scheme-handler/http $ xdg-mime default browser.desktop x-scheme-handler/https ´´´

Para usar ese programa en ´emacs´ hay que agregar a la sección ´custom-set-variables´:

´´´
 '(browse-url-browser-function (quote browse-url-xdg-open))
´´´

En ocasiones estas dos cosas no son suficientes, porque al parecer, algunos archivos .desktop agregan una InitialPreference= que mientras mayor sea, se selecciona por encima de lo que uno configura.

En ese caso, hay que crear un archivo .desktop en $HOME/.local/share/applications que contenga una preferencia mayor que la aplicación que se está usando por defecto.

Otro programa que usa expresiones regulares para decidir con qué abrir el programa es soap

Preguntar a xdg-mime programa por defecto ↑

$ file -i archivo
mime/type
$ xdg-mime query default mime/type
programa.desktop
