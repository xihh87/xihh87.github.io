Varias distribuciones de GNU/Linux ↑

Daniel Bernstein propone usar un esquema llamado slashpackage (/package) para evitar problemas de incompatibilidad entre versiones.

Este programa debería ayudar a hacer paquetes para varios sistemas

En conda se pueden construir paquetes independientes de la distribución

Ubuntu tiene este sistema supuestamente multidistribución

RedHat tiene flatpack para distribuir aplicaciones

docker sirve para utilizar software en diferentes distribuciones

AppImage permite empaquetar aplicaciones en un único ejecutable y tiene este repositorio de imagenes.
