Requisitos previos ↑

  1. Instalar ctfcli.

  2. Generar un token de administación (instrucciones).

    En este ejemplo nuestro token es: 000000000000000

    El tuyo sería diferente y mucho más largo.

  3. Configurar ctfcli:

    ctf init

    Y agregar los datos:

    Please enter CTFd instance URL: https://cschs.ctfd.io/

Please enter CTFd Admin Access Token: Please enter CTFd Admin Access Token: 000000000000000
Do you want to continue with https://cschs.ctfd.io/ and 000000000000000 [y/N]: y
Reinicializado el repositorio Git existente en /ruta/a/juegos/

Generar retos ↑

Generar un reto nuevo:

ctf challenge new [ web | crypto | programming | binary ]

Modificar el reto para mostrar lo que queremos.

Retos con contenedores ↑

Cuando se requieren contenedores para los juegos:

  1. Generar un servicio en ctfd.

  2. Etiquetar la imagen que queremos usar de acuerdo al nombre del servicio.

  3. Subir la imagen al servicio activo.
