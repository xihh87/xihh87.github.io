Si se graba únicamente el audio de la entrevista ↑

  1. Convertir el audio a video aceptado por YouTube:

    ffmpeg \
    -r 1 \
    -loop 1 \
    -i ${IMAGE} \
    -i ${AUDIO} \
    -acodec copy \
    -r 1 \
    -shortest \
    -vf scale=1280:720 \
    ${NAME}.flv
