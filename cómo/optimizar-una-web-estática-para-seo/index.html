Search Engine Optimization (SEO) ↑

Google ofrece herramientas para mejorar tus resultados de búsqueda.

Algunas de las cosas que puedes usar son:

Usar títulos descriptivos, incluyendo la marca, usar la descripción para generar los snippets de google.

  • Google tiene un manual de buenas prácticas para los metadatos de la página

Google utiliza un algoritmo para clasificar las páginas según su calidad.

Hay personas que proponen usar palabras clave de la larga cola para mejorar la conversión.

Qué tanto ganas con embudos de venta depende directamente de cuántas personas capturas con SEO.

Malas tácticas que la gente intenta a veces:

  • Usar enlaces antinaturales
  • Usar sinónimos en espera de que sean captadas por los buscadores.
  • Intentar engañar al buscador en vez de generar valor.

Google tiene un sistema para evaluar la calidad de la página web y hay herramientas para medir la usabilidad.
