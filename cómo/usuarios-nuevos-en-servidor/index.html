Instrucciones en texto para quienes no quieren esperar ↑

  1. Escribir el comando para entrar al servidor

    USUARIO=a2229727
SERVIDOR= # p.e. haase.mx, sirio.sis.uia.mx, 200.13.106.60
ssh ${USUARIO}@${SERVIDOR}

  2. Escribir tu contraseña para entrar al servidor.

    # tu contraseña es holausuario
# con los datos del paso anterior: holaa2229727
hola${USUARIO}

  3. Escribir nuevamente tu contraseña para iniciar el cambio de contraseña.

    hola${USUARIO}

  4. Escribir dos veces tu nueva contraseña.

    • La primera vez es para generar la contraseña.

    • La segunda vez es para confirmar que la escribiste bien.

  5. Entrar de nuevo al sistema con tu nueva contraseña.

    ssh ${USUARIO}@${SERVIDOR}
