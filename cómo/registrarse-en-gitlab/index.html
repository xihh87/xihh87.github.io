Cómo subir el código de forma segura sin usar tu contraseña ↑

Si usas GitLab descargas tu código desde una plataforma web, cuando quieres subir código a tu proyecto te pide usuario y contraseña.

Para ahorrarte la contraseña, puedes:

  1. Generar un par de llaves SSH

    ssh-keygen -t ed25519 -f ~/.ssh/gitlab

    Esto genera una clave privada ~/.ssh/gitlab y una clave pública ~/.ssh/gitlab.pub.

  2. Agregarla la clave pública en tu perfil de GitLab

  3. Configurar tu SSH para usar la clave pública al conectarse a GitLab:

    Host	gitlab
HostName	gitlab.com

Host	gitlab.com
User	git
RequestTTY	no
IdentityFile ~/.ssh/gitlab

Ahora toca disfrutar que no pide usuario y contraseña.
