Effective DevOps with AWS

El libro muestra cómo se usan varios servicios de AWS para:

  • Aprovisionar (troposphere, CloudFormation, awscli)
  • Configurar (Ansible)
  • CI/CD (Jenkins)
  • Monitorear y medir todo, establecer alarmas (ELK: -> Elastic Search, Logstash, Kibana).
  • Administrar los permisos (awacs)

Demuestra el proceso generando una aplicación virtual de nodejs que únicamente muestra «Hello World» y lanza monitoreo.

Me parece que no demuestra bien el CI/CD porque los comandos para actualizar las aplicaciones son manuales. Tampoco muestra cómo hacer pruebas unitarias.

Es un buen libro práctico porque explica paso por paso cómo hacer cada cosa.
