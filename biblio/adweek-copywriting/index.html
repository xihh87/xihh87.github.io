Notas acerca de «The Adweek Copywriting» por Joseph Sugarman

El objetivo del anuncio es que la gente compre. La distribución del anuncio es para que lean la primera oración.

La primera oración debe ser sencilla.

Cada oración debe llevarte a querer leer la próxima, puedes usar semillas de curiosidad.

El anuncio debe provocar una respuesta emocional y luego debe entregar razones racionales para comprar.

El anuncio debe ser personal, como si fuera un vendedor 1 a 1. Además debe responder las preguntas que la persona tiene para comprar.

El anuncio debe ser suficiente largo para cubrir el tema pero lo suficientemente corto para mantener el interés. (Como una minifalda.)

Al editar hay que mantener el mensaje y la emoción pero con menos palabras.

Antes de escribir, relajar la idea un rato. Después de escribir, dejar el texto descansar.

Importante revisar para eliminar errores.

Recomendaciones de edición:

  • Quitar hasta «que».
  • Variar el ritmo.
  • Combinar oraciones
  • Eliminar palabras innecesarias.
  • Re-arreglar para que fluyan mejor.
