Notas acerca de «The Code Book»

El libro explora las bases de los algoritmos de cifrado monoalfabéticos, polialfabéticos y la utilización de algoritmos mecánicos para cifrar cambiando el código cada letra. También muestra cómo el análisis estadístico de los mensajes puede usarse para romper estos cifrados. Finalmente ofrece un poco acerca de la historia de las claves públicas pero sin elaborar en cómo funcionan.
